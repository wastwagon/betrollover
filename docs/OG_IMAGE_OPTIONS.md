## How to Use Your Choice

1. **Preview** each option at:
   - `https://betrollover.com/og-image-option-1.png`
   - `https://betrollover.com/og-image-option-2.png`
   - `https://betrollover.com/og-image-option-3.png`
   - `https://betrollover.com/og-image-option-4.png`

2. **Replace** the current OG image:
   ```bash
   cp web/public/og-image-option-X.png web/public/og-image.png
   ```
   (Replace `X` with 1, 2, 3, or 4)

3. **Or** update `web/app/layout.tsx` to use the chosen file directly:
   ```ts
   images: [{ url: '/og-image-option-3.png', ... }]
   ```
